! 1. ID Mismatch
Do not admit if name on ID does not strictly match registration. Escalate to site manager for verification.
! 2. Mobile Devices
Zero Tolerance. If a device is seen or heard, terminate exam immediately. File incident report.
3 Muttering / Self-Talk
Issue soft warning first. Only revoke if clearly communicating with a third party.
! 4. Suspicious Eyes
Recurring looks to lap/wall/wrists = Red Flag. Warn once, then inspect area.
5 Unscheduled Breaks
Exam timer continues running. Full check-in required upon return.
6 Comfort Aids
Tissues/Sweaters allowed but must be inspected beforehand.
7 Tech Freeze / Latency
Do not close exam. Attempt refresh. If >10 mins, offer reschedule.
! 8. Note Taking
Erasable board only. Writing on hands/tissues = Prohibited.
! 9. Disruptive Behavior
Aggression overrides exam security. Dismiss immediately. Call security.
10 Emergency Protocols (Fire/Safety)
Candidate Safety First. Instruct candidates to leave ALL items at desk. Evacuate immediately. Seal room if safe.
