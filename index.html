Standard Operating Procedures & Policy Guide
PEARSON
VUE
!1. ID Mismatch
Do not admit if name on ID does not strictly match registration (excluding minor allowed variations). Escalate to site manager for secondary ID verification.
!2. Mobile Devices
Zero Tolerance. If a device is seen or heard (even if unused), terminate exam immediately. File incident report: "Prohibited Item".
3Muttering / Self-Talk
Issue soft warning first: "Please remain silent." Only revoke if clearly communicating with a third party.
!4. Suspicious Eyes
Recurring looks to lap/wall/wrists = Red Flag. Warn once. If it persists, pause exam and inspect the area for notes.
5Unscheduled Breaks
Exam timer continues running. Candidate cannot access locker/bag. Perform full check-in procedure upon return.
6Comfort Aids
Tissues/Sweaters allowed but must be inspected beforehand. If brought in mid-exam, pause session to inspect immediately.
7Tech Freeze / Latency
Do not close exam. Attempt refresh/restart. If interruption >10 mins, offer reschedule option and file case report.
!8. Note Taking
Use provided erasable board only. Writing on hands/tissues = Prohibited. Removing notes from room = Theft.
!9. Disruptive Behavior
Shouting/Aggression overrides exam security. Dismiss immediately. Call security if safety is compromised.
10 Emergency Protocols (Fire/Safety)
Candidate Safety First. Instruct candidates to leave ALL items (IDs, Keys) at desk. Evacuate immediately. Seal room if safe to do so. File "Site Emergency" report once safety is restored.
