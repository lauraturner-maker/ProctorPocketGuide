Standard Operating Procedures & Policy Guide
PEARSON
VUE
!1. ID Mismatch
Do not admit if name on ID does not strictly match registration. Escalate to site manager for secondary ID verification.
!2. Mobile Devices
Zero Tolerance. If a device is seen or heard, terminate the exam immediately. File incident report: "Prohibited Item".
3Muttering / Self-Talk
Issue soft warning first. Only revoke if clearly communicating with a third party. "Please remain silent."
!4. Suspicious Eyes
Recurring looks to lap/wall/wrists = Red Flag. Warn once. If it persists, pause exam and inspect the area.
5Unscheduled Breaks
Exam timer continues running. Candidate cannot access locker/bag. Must do full check-in upon return.
6Comfort Aids
Tissues/Sweaters allowed but must be inspected. If brought in mid-exam, pause session to inspect immediately.
7Tech Freeze
Do not close exam. Attempt refresh/restart. If >10 mins, offer reschedule option and file case report.
!8. Note Taking
Use erasable board only. Writing on hands/tissues = Prohibited. Removing notes from room = Theft.
!9. Disruptive Behavior
Shouting/Aggression overrides exam security. Dismiss immediately. Call security if safety is compromised.
10. Emergency Protocols (Fire/Safety)
Safety First. Instruct candidates to leave ALL items (IDs, Keys) at desk. Evacuate immediately. Seal room if safe. File "Site Emergency" report later.
